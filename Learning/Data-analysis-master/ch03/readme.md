
### 1. 分类模型评估
#### 二分类
- 正确率: (TP+TN)/(TP+TN+FN+FP)  
- 召回率: TP/(TP+FN)  
- F值: 2×召回率×(TP+FN)/(正确率+召回率)  
- 精准率: TP/(TP+FP)  
- 错误接受率: FP/(FP+TN)  
- 错误拒绝率: FN/(TP+FN)  
- TP+TN+FP+FN = 样本总数
#### 多元混淆矩阵    
- 正确率: (TP+TN)/(TP+TN+FN+FP)  
- 召回率，F值: 与二分类计算方法有所不同  
##### 分类模型评估方法还有ROC和AUC，增益图和KS图
### 2. 回归模型评估
- MAE: 残差绝对值的平均值
- MSE: 残差平方的平均值
- RMSE: MSE开平方
- 决定系数 R2 = SSR/SST
### 3. 聚类模型评估
- RMS
- 轮廓系数
### 4. 关联模型评估
- 支持度
- 置信度
- 提升度
